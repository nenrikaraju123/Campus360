# Campus360 Static Demo

This folder is a standalone static demo page for sharing Campus360 with others.

## Files

- `index.html` - static demo page
- `styles.css` - page styling
- `.nojekyll` - keeps GitHub Pages from applying Jekyll processing

## Deploy On GitHub Pages

1. Create a new GitHub repository, for example `campus360-demo`.
2. Upload the contents of this `static-demo` folder to the repository root.
3. In GitHub, open `Settings` -> `Pages`.
4. Select `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save.

Your public URL will usually be:

```text
https://<your-github-username>.github.io/campus360-demo/
```
